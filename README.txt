Это самый первый, сделанный мною, сайт. На тот момент я только закончил изучение HTML и мало чего понимал в CSS, но посмотрев пару видеоуроков на ютуб, я узнал о фреймворке Bootstrap, и использовал его на этом сайте
Сам сайт представляет из себя небольшую вики-страницу о персонаже из аниме "Блич" - Шихоин Йоруичи. Он состоит из двух страниц. На первой главное изображение и краткое описание, а также ссылки на главную вики страницу, а ссылка на ютуб ведет на эдит, с этим персонажем
Вторая страница состоит из фотокарточек и более подробной информации о персонаже
